# CKIS SOP Index

## What is an SOP in CKIS?

A Standard Operating Procedure (SOP) is an executable, repeatable, step-by-step procedure for accomplishing a specific operational task. See [[00-systems/sops/_convention]] for the full convention — what qualifies as an SOP, the two-tier home rule, file naming, required frontmatter, and how to add a new SOP.

## Pipeline de Onboarding

- [ ] Cliente llenó formulario / agendó llamada
- [ ] Email de confirmación recibido
- [ ] Revisado en vista "Sin categoría"
- [ ] Clasificado como Prospecto
- [ ] Evaluada factibilidad del servicio
  - [ ] **Calificado** → enviar link de llamada → mover a "Prospecto Calificado"
  - [ ] **No calificado** → enviar email explicando → mover a "Prospecto No Calificado"
- [ ] Llamada tomada y notas registradas (ver sección Notas de Llamada)
- [ ] Propuesta enviada → mover a "Negociación"
  - [ ] **Acepta** → mover a "Propuesta Aceptada" → esperar pago
  - [ ] **No acepta** → mover a "Perdido" → registrar razón → recordatorio en 30 días
- [ ] Pago recibido → mover a "Onboarding" → crear proyecto
